# kiosk-mode-react-native

A Library for setting up kiosk mode on Android with react native

## Installation

```sh
npm install kiosk-react-native
```

## Usage

```js
import { enableKioskMode, disableKioskMode } from 'kiosk-react-native';

// Enable Kiosk Mode
enableKioskMode();

// Disable Kiosk Mode
disableKioskMode();

```
