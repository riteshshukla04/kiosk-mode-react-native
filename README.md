# react-native-kiosk

A Library for setting up kiosk mode on Android with react native

## Installation

```sh
npm install react-native-kiosk
```

## Usage

```js
import { enableKioskMode, disableKioskMode } from 'react-native-kiosk';

// Enable Kiosk Mode
enableKioskMode();

// Disable Kiosk Mode
disableKioskMode();

```
